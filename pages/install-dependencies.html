Before setting up the member system, we need to install the required dependencies.

Step 1: Install bcryptjs

Run this command in your terminal from the homepage directory:

npm install bcryptjs

Step 2: Setup Database

After installing dependencies, visit:

setup-members.html

Step 3: Test the System

Once setup is complete, test the member system:

  1. Visit the trivia game
  2. Click "SUGGEST QUESTION"
  3. Sign up as a new member
  4. Submit a test question

Note: Make sure you're in the homepage directory when running npm commands!
